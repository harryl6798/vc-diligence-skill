# VC Diligence Skill for Gemini CLI

An exhaustive, enterprise-grade Venture Capital due diligence skill for the Gemini CLI. 

This skill provides a massive, multi-stage workflow covering:
- Deep technical research and search methodologies.
- Legal, regulatory, and financial risk auditing.
- Automated Investment Memorandum synthesis.
- Advanced Excalidraw visual argumentation (Architecture, Market, and Growth diagrams).

## Installation

```bash
npx skills add https://github.com/harryl6798/vc-diligence-skill
```

## Core Features
- **Phase 1: Deep Research**: Specific search patterns for Founder Alpha, Technical Moats, and Market Dynamics.
- **Phase 2: Risk Auditing**: Severity-based triage (Critical, Material, Minor).
- **Phase 3: Synthesis**: Creation of high-conviction Investment Memos.
- **Phase 4: Visualization**: Complex Excalidraw diagrams with built-in JSON syntax safety.
